ClipNotes 隐私政策

生效日期： 2025年10月22日

1. 数据收集

ClipNotes 收集您通过 ChatGPT 对话保存的笔记内容，包括：

  • 笔记文本内容
  • 标签和主题
  • 保存时间
  • 对话上下文（可选）

2. 数据存储

所有数据存储在您自己的服务器或本地计算机上。我们不会将您的数据上传到任何第三方服务器。

3. 数据使用

您的笔记数据仅用于：

  • 保存和检索您的个人笔记
  • 提供搜索和列表功能

4. 数据安全

建议您：

  • 使用强密码保护 API Token
  • 定期备份笔记数据
  • 不要在公共网络上使用

5. 数据控制

您完全控制自己的数据：

  • 可以随时删除任何笔记
  • 可以导出所有数据
  • 可以停止使用服务

6. 联系方式

如有隐私相关问题，请联系开发者。

7. 政策更新

本隐私政策可能会更新。重大变更会通过项目 README 通知。
